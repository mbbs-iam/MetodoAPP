# MetodoAPP

**What it is**
It's an app for students that will help all students, to connect them in one app. Students will share they're help and tips to work and have good degrees.

**Why I did it**
I did it because there is a lot of students that need help, but dont have the money to spend on it. And all the apps that are for students need money to work. So this app is an opensource project for students made by a student.

**For the future...**
In the future I would like that students work together using my app. I think that it will be difficult, but It's better than doing nothing
